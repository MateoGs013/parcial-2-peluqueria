# Peluquería SaaS — Parcial 2 Aplicaciones Híbridas

Proyecto fullstack MERN para la gestión de una peluquería: turnos, servicios, empleados y clientes.

## Stack

- **Backend:** Node.js + Express + MongoDB (Mongoose) + JWT
- **Frontend:** React + Vite + Tailwind CSS + React Router

## Estructura

```
parcial-2-peluqueria/
├── backend/    API REST
└── frontend/   SPA en React
```

## Cómo levantarlo

> Documentación completa en la entrega final. Por ahora, scaffolding inicial.

```bash
# Backend
cd backend
cp .env.example .env   # completar variables
npm install
npm run dev

# Frontend (en otra terminal)
cd frontend
npm install
npm run dev
```
